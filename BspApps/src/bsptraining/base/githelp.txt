GIT Repository - github - Help (BspApps\src\bsptraining\base\githelp.txt)

0) create:

* Login to https://github.com/
* Create a new repository
* NetBeans:
* "clone" it (team -> git -> clone)
* create project yes
* check in project

1) check in:

* "commit" the changes (git -> show changes -> commit)
* "push" them to the github (git -> remote -> push)

2) check out:

* [optional "revert" the changes (git -> show changes -> revert)]
* "pull" the changes from the github (git -> remote -> pull)

3) delete:

* Login to https://github.com/
* Select repository
* Settings
* Delete this repository
